# 10 SITIOS CON RSS Y CÓMO USARLOS

## ¿Qué es RSS?

Sistema que permite suscribirte a actualizaciones automáticas de sitios
web usando lectores como Feedly o Inoreader.

------------------------------------------------------------------------

1.  Google News → https://news.google.com/rss
2.  arXiv → https://arxiv.org/rss/cs
3.  GitHub Releases → https://github.com/facebook/react/releases.atom
4.  StackOverflow (tag) →
    https://stackoverflow.com/feeds/tag?tagnames=react-native
5.  Medium → https://medium.com/feed/@usuario
6.  Dev.to → https://dev.to/feed
7.  Hacker News → https://hnrss.org/frontpage
8.  Reddit → https://www.reddit.com/r/programming/.rss
9.  YouTube →
    https://www.youtube.com/feeds/videos.xml?channel_id=CANAL_ID
10. Nature → https://www.nature.com/nature.rss

------------------------------------------------------------------------

# Cómo usar un RSS (Ejemplo con Feedly)

1.  Crear cuenta en https://feedly.com
2.  Add Content
3.  Pegar URL RSS
4.  Organizar por carpetas (ML, Backend, DevOps, Research)
5.  Revisar actualizaciones centralizadas
